    配置插件 #

    1.1 新建插件 #

    打开超音数首页，点击"插件管理" -> “新建插件”，即可新建插件，进行配置。

    1.2 编辑插件 #

    如果需要对现有插件内容进行编辑, 可以通过点击"插件市场", 对以下字段(主题域、插件名称、函数描述、插件类型)进行查询，然后选择自己需要的插件右侧的"编辑"选项, 即可对插件录入的所有信息进行编辑修改。

    如果需要删除某一个插件, 直接点击对应插件右侧的"删除"即可。
