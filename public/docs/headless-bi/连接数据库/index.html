    连接数据库 #

    1. 当前支持的数据类型 #

    1. 所有支持MySQL协议的数据库，如MySQL，Doris，StarRocks等
    2. Clickhouse，PostgreSQL，H2
    3. 支持灵活扩展，欢迎贡献其它数据库类型

    2. 在SuperSonic中创建数据库连接的步骤 #

    1. 填写连接信息
    2. 点击测试连接, 若连接测试通过, 则可点击保存。否则, 返回步骤1
    3. 点击保存。
